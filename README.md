## Installation

### Clone

- Clone this repo to your local machine using `https://github.com/dennisagarwal/expense-application.git`

### Setup

 **1.** Navigate to the express-application folder in your terminal and install npm like this:

```
$ npm install
```
### Launch application

```
$ npm start
```
